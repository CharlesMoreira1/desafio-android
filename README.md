# Desafio Concrete

Criar um aplicativo usando a api do próprio Github.

# Tecnologias utilizadas

- Arquitetura MVVM
- Android X
- Kotlin
- Retrofit
- OkHttp
- Coroutines
- LiveData
- Lifecycle
- ViewModel
- Koin
- Glide
